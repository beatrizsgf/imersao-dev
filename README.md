# Projeto: 
Lab de Conexões de Habilidades

## Descrição
O Lab de Conexões de Habilidades é um projeto focado em promover a troca de conhecimento intergeracional entre todas as gerações para evitar a perda de saberes valiosos.
Ele facilita o compartilhamento de experiências e habilidades em três áreas principais:
Temas do Cotidiano: Como receitas de família ou técnicas de negociação baseadas na experiência.
Hobbies: Habilidades e práticas pessoais.
Tecnologia: Domínio de ferramentas digitais e inovações atuais.
O objetivo é simples: transformar experiências em aprendizado compartilhado, conectando diferentes gerações para que todos possam crescer juntos.

## Tecnologias Utilizadas
*   Linguagem de programação: Javascript

## Pré-requisitos
*   Github

## Instalação
Siga estes passos para configurar o projeto em sua máquina local.

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/beatrizsgf/imersao-dev
    ```

2.  **Acesse a pasta do projeto:**
    ```bash
    cd repo
    ```

3.  **Instale as dependências:**
    ```bash
    npm install
    ```

## Como usar
Aqui estão alguns exemplos de como utilizar o projeto.

1.  **Para rodar o servidor de desenvolvimento:**
    ```bash
    npm start
    ```

2.  **Para construir a versão de produção:**
    ```bash
    npm run build
    ```
